# Inventory-Dashboard
Inventory Performance &amp; BOM Dashboard using Excel, Python and Power BI
## Project Overview
Built an inventory analytics dashboard for IIoT/electronics components to monitor stock levels, reorder risk, supplier contribution, and device readiness.

## Tools Used
- Excel
- Python (Pandas)
- Power BI

## Key Features
- Low stock alerts
- Inventory value tracking
- Category-wise stock analysis
- Supplier-wise inventory value
- BOM mapping for device readiness

## Business Impact
Helps improve procurement planning, reduce stockouts, and increase inventory visibility.

- Dataset
